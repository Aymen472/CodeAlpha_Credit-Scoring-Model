# Credit Scoring Model Using Machine Learning

## Project Overview
This project predicts whether a customer is financially risky or safe using machine learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Algorithm
- Random Forest Classifier

## Dataset
German Credit Dataset

## Features
- Data preprocessing
- Missing value handling
- Label encoding
- Model training
- Accuracy evaluation
- Confusion matrix
- Data visualization

## Output
The model predicts customer credit risk.

## Result
The model successfully classifies good and bad credit risk customers.
